DNA to Protein Translation Using Python

Overview :This project translates a DNA sequence into a protein sequence using Python.

Objective: To read a DNA sequence from a FASTA file and convert it into a protein sequence.

Files
- dna_to_protein.py
- sample.fasta

Tools Used
- Python
- Visual Studio Code
- FASTA File Format

Workflow

DNA Sequence → Codons → Amino Acids → Protein Sequence

Sample Output
DNA Sequence:
ATGGCCATTGTAATGGGCCGCTGAAAGGGTGCCCGATAG
Protein Sequence:
MAIVMGR*XXAXX
